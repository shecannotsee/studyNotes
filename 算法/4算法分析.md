1.两个主要任务=正确性（不变性*单调性）+复杂度

    不变性：每一次基本操作都是同样的操作
    单调性：问题规模会在基本操作之后缩减
    正确性：经过至多躺基本操作后，算法必然终止，且能给出正确解答

2.C++等高级语言的基本指令，均等效于常数条RAM的基本指令；在渐近意义下，二者大体相当

3.复杂度分析的主要方法

    ・迭代：级数求和
    ・递归：递归跟踪+递归方程
    ・猜测加验证

3.算术级数：与末项平方同阶

    T(n) = 1+2+...+n = n(n+1)/2 = O(n^2)

4.幂方级数：比幂次高出一阶

    ・T(n) = 1^2 + 2^2 + 3^2 + ... + n^2 = n(n+1)(2n+1)/6 = O(n^3)
    ・T(n) = 1^3 + 2^3 + 3^3 + ... + n^3 = n^2 (n+1)^2 /4 = O(n^4)
    ・T(n) = 1^4 + 2^4 + 3^4 + ... + n^4 = n(n+1)(2n+1)(3n^2 +3n-1)/30 = O(n^5)
    ・......

5.几何级数（a>1）：与末项同阶

    ・T(n) = a^0 + a^1 + a^2 + ... + a^n = (a^n+1 -1)/(a-1) = O(a^n)
    ・1+2+4+...+2^n = 2^n+1 -1 = O(2^n+1) = O(2^n)

6.收敛级数：O(1)

    ・1/1/2 + 1/2/3 + 1/3/4 + 1/(n-1)/n = 1-1/n = O(1)
    ・1 + 1/2^2 + ... + 1/n^2 < 1 + 1/2^2 + ... = (3.1415926...)^2 /6 = O(1)
    ・1/3 + 1/7 + 1/8 + 1/15 + 1/24 + 1/26 + 1/31 + 1/35 +... = 1 = O(1)

7.可能未必收敛，然而长度有限

    ・调和级数：h(n) = 1 + 1/2 + 1/3 + ... + 1/n = θ(logn)
    ・对数级数：log1+log2+log3+...+logn = log(n!) = θ(nlogn)
