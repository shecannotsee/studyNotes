1.空间复杂度

2.为求解一个大规模的问题，减而治之可以

    1将其划分为两个子问题：其一平凡，另一规模岁间
    2分别求解子问题
    3由子问题的解，得到原问题的解

3.递归式算法分析1：递归跟踪分析

    ・检查每个递归实例：累计所需时间（调用语句本身，计入对应的子实例）
    ・其总和即算法执行时间


code:数组求和
sum ( int A[] , int n ){
    return ( n < 1 ) ? 
        0 : sum ( A , n-1 ) + A [ n-1 ] ;
}

3.递归式算法分析2：递推分析

    ・从递推的角度来看，为求解sum(A,n)，需
        递归求解规模为n-1的问题sum(A,n-1)       //T(n-1)
        再累加上A[n-1]                         //O(1)
    ・递归基：sum(A,0)                          //O(1)

    ・递推方程 T(n) = T (n-1) + O(1)            //recurrence
        T(0) = O(1)                           //base
    
    ・求解
        T(n) -n = T(n-1) - (n-1) = ...
                = T(2) - 2
                = T(1) - 1
                = T(0)
        故有T(n) = O(1) + n = O(n)
